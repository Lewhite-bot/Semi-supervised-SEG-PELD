This repository is under active development.

🚀 Getting Started
Requirements
Python ≥ 3.8
PyTorch ≥ 1.12
CUDA-enabled GPU (recommended)

Due to data privacy and licensing constraints, only a partial version of the spinal dataset is provided in this repository.
