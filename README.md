🧠 Semi-Supervised Nerve Root Segmentation for MISS

This repository provides the official implementation of our work on semi-supervised nerve root segmentation for minimally invasive spine surgery (MISS).

📦 Repository Status

⚠️ Code Availability

This repository is under active development.

🚀 Getting Started
Requirements
Python ≥ 3.8
PyTorch ≥ 1.12
CUDA-enabled GPU (recommended)

🧪 Dataset

Due to data privacy and licensing constraints, only a partial version of the spinal dataset is provided in this repository.
